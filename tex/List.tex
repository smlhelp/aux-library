This documentation covers the behavior of the \code{List} module -- part of the SMLNJ Basis -- as it appears in SMLNJ v110.99. Other versions of SMLNJ may have slightly different versions of \code{List}. All the values and types below are used as if the \code{List} structure has been opened. If this is not the case, then they must be prefaced with \code{List.} -- except for
\begin{itemize}
    \item the datatype \code{list}
    \item \code{map}
    \item \code{@}
    \item \code{foldl}
    \item \code{foldr}
    \item \code{rev}
    \item \code{length}
    \item \code{hd}
    \item \code{tl}
    \item \code{null}
    \item the exception \code{Empty}
\end{itemize}
which are all available at top-level.
